 # LinuxInfo
 a simple tool to do reconnaissance on a linux system


 Requirements: <br/>

 <li> python 3.7 </li>
 <br/>

 ## Usage
 Basic usage:

 ```bash
  git clone https://github.com/Br34kC0d8/LinuxInfo
  python3 app.py  
  ```

 ## Main features

 * Detect network connection
 * Kill process
 * Block Connections
 * User Info
 * OS info and kernel info
 * "Cover your Tracks"
